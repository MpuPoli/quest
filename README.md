# Этап квеста

Вы успешно собрали QR-код и попали в репозиторий.

Теперь найдите нужный коммит.

## Двоичный код

```text
101111101101001010110010
```

## Подсказка

Число майя = 16.

Переведите двоичное число в шестнадцатеричную систему счисления.  
Полученный результат — первые 6 символов хэша нужного коммита.

В истории коммитов найдите коммит, хэш которого начинается с полученного значения.
